Grammar Cheat Sheets

Present Tenses
Simple Present
(+) S + V1
(-) S + do/does not + V1
(?) Do/Does + S + V1?
She plays football.
habits • facts
Present Continuous
(+) S + am/is/are + V-ing
(-) S + am/is/are not + V-ing
(?) Am/Is/Are + S + V-ing?
They are studying now.
happening now • temporary
Present Perfect
(+) S + have/has + V3
(-) S + have/has not + V3
(?) Have/Has + S + V3?
I have finished my homework.
result now • experience
Present Perfect Continuous
(+) S + have/has been + V-ing
(-) S + have/has not been + V-ing
(?) Have/Has + S + been + V-ing?
She has been waiting for an hour.
duration • still happening
Past Tenses
Simple Past
(+) S + V2
(-) S + did not + V1
(?) Did + S + V1?
I went home yesterday.
finished time • past
Past Continuous
(+) S + was/were + V-ing
(-) S + was/were not + V-ing
(?) Was/Were + S + V-ing?
They were watching TV.
in progress • past
Past Perfect
(+) S + had + V3
(-) S + had not + V3
(?) Had + S + V3?
She had left before I came.
earlier past • sequence
Past Perfect Continuous
(+) S + had been + V-ing
(-) S + had not been + V-ing
(?) Had + S + been + V-ing?
He had been working all day.
duration before past
Future Tenses
Simple Future (will)
(+) S + will + V1
(-) S + will not + V1
(?) Will + S + V1?
I will call you later.
decision now • promise
Be Going To
(+) S + am/is/are + going to + V1
(-) S + am/is/are not + going to + V1
(?) Am/Is/Are + S + going to + V1?
She is going to visit her aunt.
plan • prediction
Future Continuous
(+) S + will be + V-ing
(-) S + will not be + V-ing
(?) Will + S + be + V-ing?
I will be studying at 8 p.m.
in progress • future
Future Perfect
(+) S + will have + V3
(-) S + will not have + V3
(?) Will + S + have + V3?
They will have finished by noon.
completed before time
Singular & Plural
cat → cats
bus → buses
baby → babies
child → children
man → men
